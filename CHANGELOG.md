# Change Log

All notable changes to the "custom-tunnel" extension will be documented in this file.

## [1.0.0] - 2026-01-29

### 초기 릴리즈

#### 추가된 기능

- 🌐 로컬 서버를 공개 URL로 터널링
- 🎯 VSCode 사이드바에서 터널 관리
- 📋 URL 클립보드 복사 기능
- 🔗 브라우저에서 바로 열기
- ⚡ 명령 팔레트를 통한 빠른 시작/중지
- 🖥️ 실시간 원격 콘솔 모니터링
- 🔍 로그 레벨별 필터링 (LOG, INFO, WARN, ERROR)
- 🔎 키워드 검색으로 로그 필터링
- 📱 QR 코드 생성으로 모바일 접속 지원
- ⏳ 서버 Wake-up 진행 상황 표시
- 🔒 HTTPS 로컬 서버 지원

#### 지원 플랫폼

- Windows, macOS, Linux

#### 요구사항

- Visual Studio Code 1.74.0 이상
- 인터넷 연결 (터널 서버 통신용)
